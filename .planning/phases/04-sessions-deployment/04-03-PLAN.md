# Deploying to Vertex AI Agent Engine

This guide covers deploying your travel booking agent to production using Vertex AI Agent Engine. In the workshop, you'll see a demonstration; this guide is for post-workshop exploration.

**Estimated time:** 30-45 minutes (first deployment)

## What is Vertex AI Agent Engine?

Vertex AI Agent Engine is Google Cloud's managed runtime for ADK agents. It provides:

- **Managed infrastructure**: No container management, auto-scaling
- **Production features**: Session/memory persistence (GA), monitoring, logging
- **API access**: Shareable endpoint for integrations
- **ADK compatibility**: Same agent code runs locally and deployed

## Prerequisites

Before deploying, you need:

### 1. Google Cloud Project (Required)

- Create or use existing GCP project
- Enable billing (Agent Engine is a paid service)
- Note your Project ID (not project number)

### 2. Enable Required APIs

```bash
gcloud services enable aiplatform.googleapis.com
gcloud services enable storage.googleapis.com
gcloud services enable cloudresourcemanager.googleapis.com
```

### 3. Create Staging Bucket

Agent Engine needs a GCS bucket for code staging:

```bash
# Replace YOUR_PROJECT and YOUR_BUCKET
gsutil mb -l us-central1 gs://YOUR_BUCKET

# Verify
gsutil ls gs://YOUR_BUCKET
```

### 4. IAM Permissions

Your account needs:
- `roles/aiplatform.user` - Deploy and manage agents
- `roles/storage.admin` - Write to staging bucket

```bash
# Grant permissions (replace YOUR_EMAIL)
gcloud projects add-iam-policy-binding YOUR_PROJECT \
  --member="user:YOUR_EMAIL" \
  --role="roles/aiplatform.user"
```

## Deployment Steps

### Step 1: Prepare Your Agent

Your agent code should be importable:

```python
# agent.py
from google.adk.agents import Agent

def create_agent() -> Agent:
    return Agent(
        model='gemini-2.5-flash',
        name='travel_booking_assistant',
        instruction='...',
        tools=[...],
    )
```

### Step 2: Create Deployment Script

See `deploy.py` for complete script. Key sections:

```python
import vertexai
from vertexai import agent_engines
from agent import create_agent

# Initialize Vertex AI
client = vertexai.Client(
    project="YOUR_PROJECT_ID",
    location="us-central1"
)

# Create agent and wrap in AdkApp
agent = create_agent()
app = agent_engines.AdkApp(agent=agent)

# Deploy (~3 minutes)
remote_agent = client.agent_engines.create(
    agent=app,
    config={
        "requirements": [
            "google-adk==1.23.0",
            "google-cloud-aiplatform[agent_engines,adk]>=1.112"
        ],
        "staging_bucket": "gs://YOUR_BUCKET",
        "display_name": "Travel Booking Workshop Agent"
    }
)

print(f"Deployed! Endpoint: {remote_agent.api_resource.name}")
```

### Step 3: Test Deployed Agent

# Query the deployed agent
async for event in remote_agent.async_stream_query(

### Step 4: Monitor and Manage

**View in Console:**

- Find your agent by display_name

**Cleanup (Important!):**

```

## Troubleshooting

### Common Errors

| Error | Cause | Solution |
|-------|-------|----------|
| `staging_bucket validation failed` | Bucket doesn't exist or wrong permissions | Create bucket, verify IAM |
| `APIs not enabled` | Required APIs disabled | Run `gcloud services enable` commands |
| `Revision not ready` | Deployment in progress | Wait 2-3 minutes, retry |
| `Permission denied` | Missing IAM roles | Add aiplatform.user role |

### Debugging

Check deployment status:
```python
print(remote_agent.api_resource.state)  # READY, CREATING, FAILED
```

View logs in Cloud Logging:
```bash
gcloud logging read "resource.type=aiplatform.googleapis.com/AgentEngine"
```

## Cost Considerations

Agent Engine billing includes:
- **Compute**: Per-minute while agent is deployed
- **Requests**: Per API call to the agent
- **Storage**: Staging bucket storage

**Recommendation:** Delete agents after testing to avoid ongoing charges.

```python
# Always clean up after testing
remote_agent.delete(force=True)
```

"""
Deployment Script for Vertex AI Agent Engine

This script deploys the travel booking assistant to production.
Run after completing the workshop exercises.

Prerequisites:
1. GCP project with billing enabled
2. APIs enabled (aiplatform, storage)
3. Staging bucket created
4. IAM permissions configured

See DEPLOYMENT.md for detailed setup instructions.

Usage:
    # Set environment variables
    export GOOGLE_CLOUD_PROJECT=your-project-id
    export STAGING_BUCKET=gs://your-bucket

    # Run deployment
    python deploy.py
"""

import os
import asyncio
from typing import Optional

# ============================================================
# CONFIGURATION
# ============================================================

PROJECT_ID = os.environ.get('GOOGLE_CLOUD_PROJECT')
LOCATION = os.environ.get('GOOGLE_CLOUD_LOCATION', 'us-central1')
STAGING_BUCKET = os.environ.get('STAGING_BUCKET')
DISPLAY_NAME = os.environ.get('AGENT_DISPLAY_NAME', 'Travel Booking Workshop Agent')

# Validate required config
if not PROJECT_ID:
    raise ValueError("GOOGLE_CLOUD_PROJECT environment variable required")
if not STAGING_BUCKET:
    raise ValueError("STAGING_BUCKET environment variable required (e.g., gs://your-bucket)")


# ============================================================
# DEPLOYMENT
# ============================================================

def deploy_agent() -> str:
    """
    Deploy the travel booking assistant to Vertex AI Agent Engine.

    Returns:
        Endpoint name for the deployed agent
    """
    import vertexai
    from vertexai import agent_engines
    from agent import create_agent

    print(f"Deploying to Vertex AI Agent Engine...")
    print(f"  Project: {PROJECT_ID}")
    print(f"  Location: {LOCATION}")
    print(f"  Staging bucket: {STAGING_BUCKET}")
    print()

    # Initialize Vertex AI client
    client = vertexai.Client(project=PROJECT_ID, location=LOCATION)

    # Create agent (from agent.py)
    agent = create_agent()
    print(f"Created agent: {agent.name}")

    # Wrap in AdkApp for deployment
    app = agent_engines.AdkApp(agent=agent)
    print("Wrapped agent in AdkApp")

    # Deploy to Agent Engine
    print("\nDeploying... (this takes ~3 minutes)")
    remote_agent = client.agent_engines.create(
        agent=app,
        config={
            "requirements": [
                "google-adk==1.23.0",
                "google-cloud-aiplatform[agent_engines,adk]>=1.112",
            ],
            "staging_bucket": STAGING_BUCKET,
            "display_name": DISPLAY_NAME,
        }
    )

    endpoint = remote_agent.api_resource.name
    print(f"\n{'='*60}")
    print(f"DEPLOYMENT SUCCESSFUL!")
    print(f"{'='*60}")
    print(f"Endpoint: {endpoint}")
    print(f"\nView in Console:")
    print(f"  https://console.cloud.google.com/vertex-ai/agent-builder")
    print()

    return endpoint


async def test_deployed_agent(endpoint: str):
    """
    Test the deployed agent with a sample query.

    Args:
        endpoint: Agent Engine endpoint from deployment
    """
    import vertexai
    from vertexai import agent_engines

    print(f"\nTesting deployed agent...")

    # Connect to deployed agent
    client = vertexai.Client(project=PROJECT_ID, location=LOCATION)
    remote_agent = client.agent_engines.get(endpoint)

    # Test query
    test_query = "Find me flights from San Francisco to Tokyo on March 15, 2026"
    print(f"Query: {test_query}")
    print("-" * 40)

    async for event in remote_agent.async_stream_query(
        user_id="test_user",
        message=test_query
    ):
        if event.is_final_response():
            print(f"Response: {event.content.parts[0].text}")
            break

    print("\nTest complete!")


def cleanup_agent(endpoint: str):
    """
    Delete the deployed agent to stop billing.

    Args:
        endpoint: Agent Engine endpoint to delete
    """
    import vertexai

    print(f"\nDeleting deployed agent...")

    client = vertexai.Client(project=PROJECT_ID, location=LOCATION)
    remote_agent = client.agent_engines.get(endpoint)
    remote_agent.delete(force=True)

    print("Agent deleted successfully!")
    print("(Billing will stop for this agent)")


# ============================================================
# MAIN
# ============================================================

def main():
    """Main entry point for deployment script."""
    import argparse

    parser = argparse.ArgumentParser(description="Deploy travel booking agent")
    parser.add_argument(
        "--action",
        choices=["deploy", "test", "cleanup"],
        default="deploy",
        help="Action to perform"
    )
    parser.add_argument(
        "--endpoint",
        help="Agent endpoint (required for test/cleanup)"
    )
    args = parser.parse_args()

    if args.action == "deploy":
        endpoint = deploy_agent()
        print(f"\nTo test: python deploy.py --action test --endpoint {endpoint}")
        print(f"To cleanup: python deploy.py --action cleanup --endpoint {endpoint}")

    elif args.action == "test":
        if not args.endpoint:
            raise ValueError("--endpoint required for test action")
        asyncio.run(test_deployed_agent(args.endpoint))

    elif args.action == "cleanup":
        if not args.endpoint:
            raise ValueError("--endpoint required for cleanup action")
        cleanup_agent(args.endpoint)


if __name__ == "__main__":
    main()
